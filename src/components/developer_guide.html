OpenParking: Developer Guide

OpenParking is an Android application that provides a platform from which people can share parking spaces with others.

Features include:

  1. Open marketplace where users and contributors can list and purchase parking spots
  2. List and map views using location information
  3. Physical or electronic transaction option
  4. Review/rating system available for all transactions

Want to work with OpenParking's code? Great! Let's get you started.

You can get the source code an a previously compiled .apk on our github repository.

To get the source from GitHub:

https://github.com/open-parking/OpenParking

The .apk is located in OpenParking Code/app/debug/OpenParking.apk.

You will need a way to build and compile Java programs for an Android device; one of the easiest ways is using Android Studio.
Open the project in Android Studio, and it should automatically build and make gradle dependencies.

If you experience problems building the project, you should consider having these versions of the dependencies.

You will also need these things:

  1. A Google Maps API key
  2. A Google Firebase project
  3. A Paypal developer account

Setup instructions:

  1. Insert your API key into OpenParking Code/app/src/debug/res/values/google_maps_api.xml
  2. Setup your project to use Firebase. You should have a google-services.json in OpenParking Code/app/google-services.json.
    Insert your Firebase details such as app ID and API key there.
  3. Insert your Paypal credentials in OpenParking Code/app/src/main/java/com/example/openparking/Config/Config.java,
